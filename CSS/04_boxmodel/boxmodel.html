This is a block element. Block elements take up the full width available and creates a new line before and after it.

Box Model Components

The box model consists of:

  • Content: The actual content of the box, where text and images appear
  • padding: Clear an area around the content (inside the border)
  • border: A border that goes around the padding and content
  • margin: Clear an area outside the border (between the border and the other elements next to it)
